# wend-v1
edition-driven file navigation plugin for vim.

## features
### edition-driven
the navigation window like this:

```bash
drwxr-xr-x .../foo/
-rw-r--r-- .../foo
-rw-r--r-- .../foo1
```

each line is added up by A) an access control block and B) a folded path. you can edit it in INSERT mode and:

- press `<enter>` to enter a file/folder if you edited the path;
- `:w` to save the changed access control.

any navigation is led & driven by edition only.
